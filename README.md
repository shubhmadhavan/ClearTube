## How to Use
- Install a compatible userscript manager extension for your browser
  - Popular options are:
    - [Tampermonkey](https://www.tampermonkey.net/)
    - [Violentmonkey](https://violentmonkey.github.io/get-it/)
- Download the ClearTube-1.3.user.js file from this repository
- Install the ClearTube-1.3.user.js file on the Userscript manager
- Go to Youtube; click on the userscript manager extension
  - For Tampermonkey, it looks like this: 
 <img width="230" alt="2025-11-02 (09-55-37)" src="https://github.com/user-attachments/assets/5f9cbf30-ffa5-47bc-8956-b68951418a73" />
- Toggle Youtube features as per your preferences

## Features:
You can toggle viewing (Disable/ Enable) the following items from various Youtube Pages:
- Home Page Video Recommendations 
- Home Page Shorts Recommendations
- Video Recommendations
- Comments
- Sidebar
- Tags/ Categories
- Live Chat
- Donations
